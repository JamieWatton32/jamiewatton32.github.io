Natural flocking simulation (Now ported to JavaScipt!)

Similar to Conways Game of Life, Boids is a artifical life program. first developed in 1986, it simulates the flocking behaviour of birds, schools of fish, etc.

Using the p5.js library I ported my orginal python version of the algorithm into javascript so youm the reader, can check it out without any downloads!
